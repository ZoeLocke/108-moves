Welcome to 108 Moves. Here you can view the full list of simplified versions of the moves as practised by the Caerphilly beginner class of Tai Chi Evolutions. Each move is a card that can be clicked on and expanded to view step-by-step instructions for the move.

The sections between each card, showing the {{< icon type="transition" >}} icon, are reminders of the movement that transitions between each move (these moves also show in the step-by-step, so be sure not repeat them!). Transition notes are hidden from screen readers to improve the screen reader experience.

Use the **Advanced** button to toggle on and off more detailed descriptions of each step of a move.

Use the **Size** button to change the size of the cards on this screen, so you can have more visible on the screen, or have all text be easier to read. The Size button is hidden from screen readers to improve the screen reader experience.

Moves may include a {{< icon type="direction" >}} icon followed by a **direction**. This indicates which direction the move ends in; which way you will be facing once the move is complete. Directions include the walls *front* and *back*, the sides *doors* and *stage*, or combinations of wall and side, such as *front/stage*.

If a move card is green and has a {{< icon type="learned" >}}  icon after it's name, the move has been covered in class (this will also apply to moves that are further in the set than we have reached, but are repeats of moves we already know). If a move card if red and has a {{< icon type="unlearned" >}} icon after it's name, the move has been covered in class.

A small {{< icon type="file" >}} icon in the top-left corner of a move card indicates that step-by-step instructions have been added for that move. A {{< icon type="file-half-dashed" >}} icon indicates that step-by-step instructions are still being added.
