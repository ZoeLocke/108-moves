Welcome to 108 Moves. Here you can view the full list of moves practised by Tai Chi Evolutions. Each move is a card that can be clicked on and expanded to view step-by-step instructions for the move.

The sections between each card, showing the {{< transition-icon >}} icon, are reminders of the movement that transitions between each move (these moves also show in the step-by-step, so be sure not repeat them!)

Use the **Advanced** button to toggle on and off more detailed descriptions of each step of a move.

Use the **Size** button to change the size of the cards on this screen, so you can have more visible on the screen, or have all text be easier to read.

Click the arrow at the top of this box to close the box, giving you more room for the steps.
